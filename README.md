# 🌌 Landing Page com HTML e CSS

### Formação CSS Web Developer

Este projeto demonstra uma página desenvolvida como parte da Trilha de CSS oferecida pela Digital Innovation One (DIO). Ele abrange técnicas de HTML e CSS para criar uma página responsiva e estilizada.

## 🛠️ Estrutura do Projeto

Este projeto consiste em uma página web estruturada em HTML e estilizada com CSS, desenvolvido como parte da Trilha de CSS da DIO. Ele apresenta um layout organizado com header, seções de conteúdo e footer, acompanhado de elementos visuais modernos como gradientes, imagens de fundo e tipografia estilizada. O CSS é dividido em um arquivo de normalização (reset.css) e um de estilização principal (style.css), garantindo consistência e estética responsiva. O foco é criar uma experiência visualmente atrativa e funcional, alinhada às boas práticas de desenvolvimento.

### 📝 HTML

O arquivo principal do projeto é o `index.html`. Ele contém:

- **Header:** Inclui um banner com logotipo, título e descrição da trilha, acompanhado de um botão de inscrição.
- **Main:** 
  - Seção "O que vou aprender?" com descrição dos módulos do curso.
  - Seção "Transforme o mundo com a gente" com uma chamada visual.
  - Seção "Evolua e encare novos desafios profissionais" com imagem, descrição e mensagem motivacional.
- **Footer:** Contém um logotipo e um link para a página oficial da DIO.

### 🎨 CSS 

A estilização é feita através dos arquivos:

- `style.css`
- `reset.css`

#### Arquivo `reset.css`
Este arquivo redefine as margens, paddings e outras propriedades para garantir consistência entre navegadores. Inclui fontes externas:

- [Raleway](https://fonts.google.com/specimen/Raleway)
- [Open Sans](https://fonts.google.com/specimen/Open+Sans)

#### Arquivo `style.css`
Este arquivo contém a estilização principal do projeto:

- **Body:**
  - Fundo preto.
  - Fonte padrão definida no `reset.css`.

- **Banner:**
  - Fundo com imagem e gradiente linear.
  - Contéudo centralizado e estilizado com cores brancas e gradiente no título.

- **Seção "O que vou aprender?":**
  - Texto centralizado e módulos destacados com bordas, cores diferenciadas e sombreamento.

- **Seção "Transforme o mundo com a gente":**
  - Imagem de fundo fixa com texto estilizado e sombreado.

- **Footer:**
  - Gradiente suave e alinhamento central.

## 🎮 Funcionalidades

- Layout responsivo adaptado a diferentes tamanhos de tela.
- Utiliza gradientes lineares para efeitos visuais nos textos e elementos.
- Design moderno e consistente com boas práticas de estilização.

## 🛠️ Como Executar

1. Clone este repositório.
2. Certifique-se de que os arquivos estão organizados da seguinte forma:
   ```
   /src
   |-- /style
   |   |-- style.css
   |   |-- reset.css
   |
   |-- /assets
       |-- /images
           |-- logo.png
           |-- banner.png
           |-- woman-code.png
           |-- professional-challenges.png
   ```
3. Abra o arquivo `index.html` em qualquer navegador web.
